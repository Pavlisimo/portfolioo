# 💻 Personal Portfolio Website

Добро пожаловать в репозиторий моего личного сайта-портфолио! Этот проект создан для демонстрации моих навыков в веб-разработке, выполненных проектов и предоставления контактов для связи.

🔗 **[Посмотреть live-демо сайта](https://pavlisimo.github.io/portfolioo/)**

---

## ✨ Особенности / Features

* **Адаптивный дизайн:** Сайт отлично отображается на любых устройствах — от мобильных телефонов до больших мониторов.
* **Интерактивный UI:** 
  * Динамическое поведение хедера при скролле экрана (smooth header scroll).
  * Всплывающие модальные окна (modal functionality) для детального просмотра информации.
* **Чистый код:** Проект разделен на логические папки (`css/`, `js/`), что упрощает его масштабирование и поддержку.

---

## 🛠️ Стек технологий / Tech Stack

* **HTML5** — семантическая структура веб-страницы.
* **CSS3** — стилизация, сетки (Flexbox/Grid), анимации и медиа-запросы для адаптивности.
* **JavaScript (ES6+)** — логика взаимодействия, работа с DOM, обработка событий скролла и управление модальными окнами.

---

## 🚀 Деплой / Deployment

Проект автоматически собирается и публикуется с помощью сервиса **GitHub Pages**. Все изменения в ветке `main` сразу же отображаются на живом сайте.
